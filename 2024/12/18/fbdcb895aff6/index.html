准备工作

1、搭建免费VPN所需代码+优选工具：https://www.freedidi.com/17535.html

2、免费域名注册链接：https://www.freedidi.com/17434.html

操作步骤

Cloudflare创建Worker

填写名称，创建的worker项目名称最好使用系统默认的，别自定义，以免被系统识别到特殊字符而被屏蔽。

点击编辑代码

若页面一只加载不出来，可F12打开控制台，然后点击文件就能显示部分内容，直接全选粘贴

编辑worker.js代码

这个代码来自CM在【 GitHub 】社区的开源项目，代码已经内置IP优选和代理功能，自带动态的UUID，可以大大减少手动配置过程，非常适合新手和特殊用户。

加密版：【**点击获取**】推荐使用加密版，有混淆作用！

明文版：【**点击获取**】

粘贴后重新部署，然后返回上一步，点击设置，选择变量与机密

添加变量，变量名称：PASSWORD，值可自定义，点击重新部署

部署好后点击访问按钮

在地址栏添加 / ${PASSWORD} 并访问

这样就能访问到订阅地址

此时海外环境已能正常订阅使用，非海外环境需绑定自定义域名才可使用

注册自定义域名

【白嫖福利】注册US.KG免费域名

绑定自定义域名

绑定好后把刚才的网址替换为自定义域名

使用V2rayN添加订阅

导入后更新订阅（不通过代理），会更新12个节点

IP优选

优选IP时需要清除代理

CloudFlare 优选ip 和 优选域名的获取方法！懒人加速必备
