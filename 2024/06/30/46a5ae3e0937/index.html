开启IPv6

发表于2024-06-30|更新于2024-12-18|小技能
|阅读量:

一、检测是否已开启IPv6

Win + R 输入 cmd 打开命令提示符窗口

输入ipconfig，如果出现2开头的ipv6地址则已经开启

如果没有显示任何ipv6地址信息则说明没有开启Ipv6，如果仅有 fe80开头的IPv6地址则代表虽然有开通但仍需一些配置才能使用。

或者使用 IPv6地址查询工具 检测，如果显示Ipv6访问优先则代表已开启IPv6

二、开启IPv6

打开 控制面板 => 网络和Internet => 网络和共享中心 => 查看网络状态和任务 => 更改适配器设置

选择当前正在使用的网卡，右键 => 属性

勾选IPv6确定即可
