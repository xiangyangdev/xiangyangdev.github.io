实用配置

语言汉化

推荐： https://t.me/setlanguage/zh-hans-raw

备用: https://t.me/setlanguage/classic-zh-cn

安全设置

隐私设置

隐私设置不好，泄露你的个人信息，非常重要。

设置路径： 设置 -> 隐私

除了头像跟介绍，其他的能设置仅自己可见就设置仅自己可见

手机号不建议86，太容易泄露了。

两步验证及锁定码

两步验证： 非常重要，开启之后不管是手机还是邮箱验证码登录，必须要输入你的密码，这个非常重要。

锁定码：离开一段时间后，打开电报就需要输入密码，对于不设置密码的电脑刚需啊，毕竟一打开就一堆XX内容！

邮箱接码登录

非常重要，以后就不需要担心失联了，就算手机号更换了，电报邮箱接码登录依旧有效。

以后陌生的设备登录直接发送到邮箱，不会发送到手机了。

傻瓜式教程地址：https://youtu.be/LvxNd6YCIIU

解除敏感内容限制

默认情况下，电报的敏感内容显示不出来，需要去网页端手动解除

操作路径： Settings -> Privacy and Security -> Disable filtering （勾选即可）

勾选禁用敏感过滤，即可显示出来。

搜索引擎

一个好的搜索引擎，可以一件帮你找到你需要的内容或者是群组，非常重要。

极搜 堪称电报telegram目前最强搜索引擎，啥都能搜到！

分组及归档

痛点：

有没有当着其他人的面从来不敢打开电报，毕竟一打开之后首页一堆XX内容，分组及归档拯救你。

分组：创建分组，然后将群根据分类添加到分组，之后将群归档，在首页就看不到这个群了，还你一个干净的首页，以后在公开场合随便打开电脑都没关系。

分组路径：设置 - > 分组 -> 创建分组，将群归档即可！
