Warp升级Zero Trust

发表于2024-07-01|更新于2024-12-18|小技能
|阅读量:

一、获取无限流量密钥

使用Telegram机器人获取

订阅 @generatewarpplusbot机器人

使用 /generate 指令生成密钥，首次生成提示需要订阅几个频道，订阅后重新发送命令即可

每两个小时可以获取一次密钥

使用Replit程序获取

打开 CloudFlare Warp 密钥生成仓库

可以使用Google邮箱快速登录，点击Fork项目

仓库信息保持默认即可

点击运行

输入要生成的密钥数量运行即可

如果生成的密钥只有1G流量重新运行获取即可

使用NekoBox客户端生成

可参考 Warp优选IP结合NekoBox配置无限高速节点

二、开启Zero Trust

注册并登录 CloudFlare，打开 Zero Trust

输入团队名称

选择0元计划

点击进行付款

添加付款方式

使用 美国地址生成器 获取一个信用卡信息

也可以跳过此步骤，重新打开个浏览器窗口登录CloudFlare即可

可以使用谷歌翻译插件翻译下网页，点击验证

选择验证Warp

添加规则

规则名称可随便填写，可以选择邮箱或者邮箱结尾，这里选择邮箱结尾，可以输入多个，回车确认，点击节省保存

打开CloudFlare客户端 => 偏好设置 => 账户 => 使用 Cloudflare Zero Trust 登录

输入注册的团队名称

输入绑定的邮箱

输入验证码

验证成功

然后就能使用 Zero Trust 进行连接了

如果提示无法连接，可以退出登录，挂代理重新登录，可参考 解决zerotrust连接问题

zerotrust有很多人能连上了，还是有很多人连接不上，主要跟你登录账号的IP有关，直接国内IP登录账号，很大概率后续连接不上。可以随便用个国外的代理开前置代理登录，比如安卓手机上可以用v2rayng之类的开个全局代理，然后登录zerotrust账号（如果已经登录，注销了再登录），登录成功了可以试试连接正常了，之后就把代理关了，随便断开连接都没什么问题。可以看上图我国内直接登录的话几乎都是连不上的。当然了流量出口还是香港的。
