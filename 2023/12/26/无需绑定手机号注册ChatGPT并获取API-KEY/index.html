无需绑定手机号注册ChatGPT并获取API KEY

Created2023-12-26|Updated2023-12-22|工具箱
|Post Views:

一、注册

可以使用临时邮箱进行注册

临时邮箱地址 : https://www.linshiyouxiang.net

获取随机临时邮箱，可以点击新邮箱获取一个新邮箱

进入OpenAI官网 => Try ChatGPT => Sign up 进行注册

点击登录后提示邮箱不可用，莫慌莫慌

回到临时邮箱，可以看到已经收到了OpenAI的邮件

点击邮箱详情进行验证即可

点击链接会自动登录并跳转到OpenAI页面，到这里账号就注册成功了

重新用账号密码登录即可正常使用了

二、获取API KEY

注册成功并登录之后新打开个标签页打开：https://platform.openai.com/

打开浏览器控制台

如何打开浏览器控制台？

方法一：在键盘上按 Ctrl + Shift + I（Windows）或 alt + cmd + I（macOS）
方法二：键盘直接按 F12
方法三：在界面任意位置点鼠标右键 检查 或者 审查元素
方法四：浏览器右上角设置 -> 更多工具 -> 开发者工具/开发人员工具 等字眼

然后点击 网络(Network) -> Fetch/XHR -> 过滤框里输入 login，然后刷新页面

可以检索到一条 login的 网络请求，点开 请求 -> 预览(Preview) -> user -> session，会发现有一个 sensitive_id 的字段，对应的值就是 API Key

例如此处我的 API Key 就是 sess-hZNZwMegcl0b1CnQGMN9JXxQJQ16YqSP3Udh2PuE

然后就可以在需要 API Key 的应用或者站点使用

重新登录后API Key 会改变，其实就相当于是个 Session ID，一旦你的 API Key 不小心泄露或者失效，只需退出，重新登录下 OpenAI 的官网，重新按上面方式打开控制台查看最新的 API Key 即可

注意，如果想刷新SessionID,需要重新登录OpenAI，而不是ChatGPT网页端
